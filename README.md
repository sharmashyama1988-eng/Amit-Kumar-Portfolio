🚀 Amit Kumar — Creative Developer Portfolio
यह मेरा पर्सनल पोर्टफोलियो है जिसे मैंने अपनी स्किल्स, प्रोजेक्ट्स और क्रिएटिविटी को दिखाने के लिए बनाया है। इसमें AI Integration, 3D Web Development, और Modern UI/UX Design का इस्तेमाल किया गया है।

🌐 Live Demo
आप इसे यहाँ लाइव देख सकते हैं: https://amit-kumar-portfolio-nine.vercel.app/

✨ Key Features
Custom Particle Background: Canvas API का उपयोग करके बनाया गया डायनामिक इंटरैक्टिव बैकग्राउंड।

Custom Cursor: डेस्कटॉप के लिए एक स्मूथ, रिंग-बेस्ड कस्टम माउस कर्सर।

Typewriter Effect: मेरी मुख्य विशेषज्ञता (AI, 3D, Full-Stack) दिखाने के लिए एनिमेटेड टेक्स्ट।

Fully Responsive: मोबाइल और टैबलेट के लिए पूरी तरह से ऑप्टिमाइज्ड।

Project Showcase: मेरे टॉप प्रोजेक्ट्स जैसे CityBuilder 3D और AI Cam Ultra का विस्तृत विवरण।

EmailJS Integration: सीधे ब्राउज़र से मुझे ईमेल भेजने के लिए वर्किंग कांटेक्ट फॉर्म।

🛠️ Tech Stack
Frontend: HTML5, CSS3 (Modern Flexbox/Grid), JavaScript (ES6+).

Libraries: Three.js (3D Elements), EmailJS (Contact Form).

Design: Figma, Adobe Suite.

Deployment: Vercel & GitHub.
